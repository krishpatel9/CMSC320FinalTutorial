Introduction

The goal of this project is to walk you through the entire Data Science Pipeline. In order to do so, we have chosen to focus on how different features (like RAM, Storage Type, Screen, etc.) and brands of laptops affect their prices. This project is particularly relevant for consumers looking to make informed decisions when purchasing a laptop, as well as for retailers and manufacturers who aim to understand market trends and pricing strategies.

Why is this Important?

In today's technology-driven world, laptops are an essential tool for both personal and professional use. Over time, the number of options available has increased significantly, with there being more laptop options now than ever. This often makes choosing the right laptop a daunting task. Consumers often struggle to balance their needs with their budgets, and understanding what drives the price of a laptop can significantly ease this decision-making process.

The hypothesis we want to test is that the brand of a laptop has a more significant influence over the price of a laptop than the actual technical specifications. This insight could reshape how consumers approach purchasing decisions and how companies market their products. If brand influence is more significant, then consumers who are impartial to brand may opt for lower-priced options with similar specs, while manufacturers might focus on brand enhancement rather than just improving technical aspects.

By analyzing laptop specifications and prices, we aim to uncover patterns that guide consumers to the best value and help manufacturers and retailers optimize their pricing strategies.

def extract_storage_info(memory):
    if '+' in memory:
        return None, None  # Discard combined storage

    size, type = memory.split()[0], memory.split()[1]

    # Convert size to GB if necessary
    if 'TB' in size:
        size = float(size.replace('TB', '')) * 1024
    else:
        size = float(size.replace('GB', ''))

    # Determine storage type
    if 'SSD' in type:
        storage_type = 'SSD'
    elif 'HDD' in type:
        storage_type = 'HDD'
    elif 'Flash' in type:
        storage_type = 'Flash'
    elif 'Hybrid' in type:
        storage_type = 'Hybrid'
    else:
        storage_type = 'Other'

    return size, storage_type

# Apply the function to the Memory column
df['Storage_Size'], df['Storage_Type'] = zip(*df['Memory'].apply(extract_storage_info))

# Drop rows with combined storage
df = df.dropna(subset=['Storage_Size', 'Storage_Type'])

# Drop the original Memory column as it is no longer needed

Model: Analysis, hypothesis testing & ML:

In this section, we plan to use various modeling and analysis techniques, such as Linear Regression, Support Vector Machines, and Random Forests to predict values outside the scope of our data.

Earlier we calculated the average price for specific features (CPU and Brand). Now we want to find a model that best fits the price, based on all of the features provided to us (Comapny, CPU, GPU, Ram, Storage, Size, etc.) To do this we will create a linear regression model, with price as the output, and all of the aforementioned categories and more, as the feature vector. We will then train a linear regression model to find the appropriate weights for each feature, and thus get a model that can predict price as a function of all of these.

Here we choose the features that we will consider when making our model. We also split up our data into training and testing data, to ensure that our model can be tested properly, while still having enough data to be trained on.

In[]:
# Define the target variable and features
X = df.drop(columns=['Price_US', 'Cpu_Model', 'Product'])

def plot_feature_weights(weights_df, title):
    plt.figure(figsize=(10, len(weights_df) * 0.5))
    sns.barplot(x='Weight', y='Feature', data=weights_df.sort_values(by='Weight', ascending=False))
    plt.title(title)
    plt.xlabel('Weight')
    plt.ylabel('Feature')
    plt.show()
